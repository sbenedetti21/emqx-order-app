## Description

[Nest](https://github.com/nestjs/nest) project using EMQX to handle events.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# Start containers for emqx and cockroachdb
$ docker compose --file compose-local.yml up -d --remove-orphans

# Run this commands on separate consoles
$ npm run start:dev
$ npm run start:dev stock
$ npm run start:dev billing

# To check the status of emqx
http://localhost:18083/#/dashboard/overview

user: admin
pass: Prueba1! (or maybe 'public')

```
